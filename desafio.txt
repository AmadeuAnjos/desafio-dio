Iniciando o desafio prático proposto pela Digital Innovation One

1º Criar repositório:
digitalone

2º Inicalizando o git no repositório:
git init

3º Adicionar o arquivo para ser "commitado"
git add . 

4º Fazendo o commit 
git commit 

Após fazer o processo acima, vou dar um push para o Github

1º git remote add origin git@github.com:AmadeuAnjos/desafio-dio.git

2º git push origin master 

Após fazer este processo verifiquei e estava ok no Github

Alterei o arquivo, adicionei e fiz um novo commit
